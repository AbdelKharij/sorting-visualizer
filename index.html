How does the Bubble Sort Work?

This algorithm works by looping through the array, comparing the starting number to the number right next to it. If the first number is larger, the algorithm makes a swap, if not, nothing happens. The code then moves to the next elements and repeats the process. After the algorithm runs through the entire array, it will have the largest number at the end. The code will keep running this through the array (minus the last number since it's already in place) until it reaches the desired result.

function bubbleSort(){

for(var i = 0; i < arrayValues.length; i++) {

for(var j = 0; j < (arrayValues.length - i - 1); j++){

if (arrayValues[j] > arrayValues[j + 1]){

var temp = arrayValues[j];

arrayValues[j] = arrayValues[j + 1];

arrayValues[j + 1] = temp;

}

}

}

}
